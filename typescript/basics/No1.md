# TS简介

## TypeScript 是什么

- 以 `JavaScript` 为基础构建的语言

- 一个 `JavaScript` 的超集

- `TypeScript` 扩展了  `JavaScript`，并添加了类型

- 可以在任何支持 `JavaScript` 的平台中执行

> [!warning] 注意
> TS 不能被 JS 解析器直接执行

## TypeScript 增加了什么

- 类型

- 支持ES的新类型

- 强大的开发工具

- 添加ES不具备的新特性

- 丰富发配置选项

## TypeScript 开发环境搭建

- 1、下载 `Node.js`

> [https://nodejs.org/zh-cn](https://nodejs.org/zh-cn)

- 2、安装 `Node.js`

- 3、使用 `npm` 全局安装 `typescript`

> 进入命令行

> 输入：`npm install -g typescript`

- 4、创建一个 `ts` 文件

- 5、使用 `tsc` 对 `ts` 进行编译

> 进入命令行

> 进入 `ts` 文件所在目录

> 执行命令：`tsc  xxx.ts`
